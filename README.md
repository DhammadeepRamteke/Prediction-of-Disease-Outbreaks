**Disease Outbreak Prediction**

This is a machine learning-powered web application for predicting disease outbreaks based on input features. The model is built using scikit-learn and deployed using Streamlit.

🚀 Live Demo

[Click here to try the app](https://prediction-of-disease-outbreaks-my-project.streamlit.app/)

📌 Features

User-friendly web interface using Streamlit.

Machine Learning model to predict disease outbreaks.

Hosted on Streamlit Community Cloud.
